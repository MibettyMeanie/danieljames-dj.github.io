Projects

Zoho Sheet

In Zoho Sheet iOS, I've worked in migration of the app from Objective-C to Swift 3, and later to Swift 5. I played a key role in introducing VIPER architecture in the app, which made it much more scalable and integrated OOP concepts in the codebase. Other key features that I implemented are Data formats, sheet listing page, landing view, charts listing, color picker view, etc.

Bothoven

Bothoven is an interesting project where 2 robots collaborate and play a given sequence of notes placed randomly on an arena to produce music. It was done as a part of e-Yantra Robotics competition conducted by IIT Bombay. The project was realised using Fire Bird V Robots. ZigBee technology was also used to enable wireless communication.

Sunshine

This hybrid app was build for the NASA Space App Challenge 2017, a 24-hour hackathon using Ionic framework. This app lets you calculate the energy produced by the solar panel system and helps you to identify which all appliances can be used with the available power. An additional feature of this app is that it can calculate tilt angle of solar panel based on season and latitude input.

Waste Disposal Management Application

Waste Disposal Management Application is a hybrid mobile application that runs on all major platforms. The main objective was to connect garbage collectors and users (similar to Uber and Ola). The app was built in Ionic framework along with Node.js and MySQL.

Medical App

It's an Android application that intends to provide faster medical care by connecting patients with nearby doctors. This app was built as a part of 24- hour hackathon conducted by Envestnet. The main tools used were Android Studio, Node.js and MongoDB.
